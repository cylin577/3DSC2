# Product Definition: 3DSC2

## Initial Concept
**3DSC2** is a 3DS Camera View & Control (TAS & Event Replay) tool to stream and control any 3DS using a camera and PC. It provides a software-based capture and control solution for Nintendo 3DS consoles, enabling streaming and remote play for all 3DS models without the need for expensive internal hardware modifications.

---

## Overview
**3DSC2** utilizes a standard webcam to "capture" 3DS screens and a network-based protocol to send PC inputs back to the console, enabling streaming and remote play for all 3DS models.

## Core Goals
- **Low-Latency Streaming:** Provide a smooth, high-performance video feed for casual players and streamers.
- **Automated Setup:** Use Computer Vision to automatically detect and calibrate the Region of Interest (ROI) for both 3DS screens.
- **Advanced Visuals:** Enhance the low-resolution 3DS feed using AI-powered Super-Resolution and stabilize the image with Dynamic Motion Tracking.
- **Seamless Control:** Map PC gamepads and mouse inputs to 3DS buttons and the touch screen via Luma3DS InputRedirection.

## Target Audience
- **Casual Players:** Users looking for an easy, cost-effective way to play 3DS games on a larger screen.
- **Streamers/Content Creators:** Providing a software alternative to hardware capture cards for O3DS models.
- **TAS/Automation Enthusiasts:** Recording and replaying inputs with millisecond precision.

## Key Features
- **Smart Calibration:** Automatic detection of top/bottom screens using feature matching or color markers.
- **Enhanced Viewport:** Separate, resizable windows for each screen with optional shaders.
- **Input Redirection:** Plug-and-play support for PC gamepads and automated touch screen scaling.
- **TAS Engine:** Frame-perfect recording and playback of console inputs.
- **AI Upscaling:** Real-time ML models to sharpen the 240p/400p 3DS output for modern monitors.
